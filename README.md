# AWS Test Website 🚀

This is a simple HTML project used to test deployment on AWS.

## Project Description

This repository contains a basic `index.html` file to verify that a website can be successfully deployed to AWS infrastructure.

The page displays a simple message confirming that the deployment is working.

## Technologies Used

* HTML
* Git
* GitHub
* AWS

## Project Structure

```
.
├── index.html
└── README.md
```

## How to Run Locally

1. Clone the repository
2. Open the `index.html` file in your browser

## Deployment Options

This project can be deployed using:

* Amazon S3 Static Website Hosting
* Amazon EC2 Web Server
* AWS Amplify

## Purpose

The purpose of this project is to perform a quick AWS deployment test and verify that the environment is correctly configured.
